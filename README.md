# HSUU — Human Systems Under Uncertainty

Programa de investigación sobre sistemas humanos que operan bajo incertidumbre.

## Tesis inicial

Los sistemas humanos expuestos a incertidumbre desarrollan mecanismos de regulación colectiva para preservar su continuidad, restaurar previsibilidad y reorganizar sus funciones.

## Objetivo

Construir un marco teórico, metodológico y aplicado para observar, modelar e intervenir dinámicas colectivas en contextos de alta incertidumbre.
